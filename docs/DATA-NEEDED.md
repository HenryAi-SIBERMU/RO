## Format CSV Standar

Kirimkan data dalam format CSV sederhana. Contoh:

```csv
tahun,total_harta,jumlah_pejabat
2019,450.5,34
2020,512.3,34
2021,489.1,34
```

Simpan di folder: `celios-streamlit/data/` dengan nama: `[nomor_halaman]_[id_gambar].csv`  
Contoh: `data/page020_gambar7.csv`
